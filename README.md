# CRUD TEST

Personal application.

## Setup

```bash
npm install
```
Next, for the database here's the script to generate the tables ([script](https://github.com/Mkultr730/CRUD-TEST/blob/master/src/sql/CRUD_create.sql))

## Usage
```bash
  npm run start:dev
```
